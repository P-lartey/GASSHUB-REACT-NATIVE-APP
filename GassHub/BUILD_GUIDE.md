### 🔧 Required Configurations:

#### 1. **Google Services (Android)**
Ensure `google-services.json` is in the project root with:
- Firebase project configuration
- Google Sign-In credentials
- Push notifications setup

#### 2. **Apple Services (iOS)**
Ensure you have:
- Apple Developer Account ($99/year)
- App Store Connect account
- Push notification certificates
- App icons and splash screens

#### 3. **EAS Build Setup**
```bash
# Install EAS CLI globally
npm install -g eas-cli

# Login to your Expo account
eas login

# Configure EAS for your project
eas build:configure
```

## 🚀 Build Commands

### Development Build (Testing)
```bash
# For Android
eas build --platform android --profile development

# For iOS
eas build --platform ios --profile development
```

### Production Build (Store Release)

#### Android (Google Play Store)
```bash
# Build Android App Bundle (.aab)
eas build --platform android --profile production

# Submit to Play Store
eas submit --platform android --latest
```

#### iOS (App Store)
```bash
# Build iOS app (.ipa)
eas build --platform ios --profile production

# Submit to App Store
eas submit --platform ios --latest
```

## 📦 Build Outputs

### Android
- **Format**: Android App Bundle (.aab)
- **Location**: Generated by EAS Build
- **Size**: Optimized for Play Store
- **Signing**: Handled by EAS/Google Play

### iOS
- **Format**: IPA file
- **Location**: App Store Connect
- **Size**: Optimized for App Store
- **Signing**: Requires Apple certificates

## 🏪 Store Submission

### Google Play Store
1. Go to [Google Play Console](https://play.google.com/console)
2. Create new app
3. Upload `.aab` file to Production track
4. Fill out store listing
5. Submit for review

### Apple App Store
1. Go to [App Store Connect](https://appstoreconnect.apple.com)
2. Create new app
3. Upload build via Xcode or Transporter
4. Fill out app information
5. Submit for review

## 🔐 Required Credentials

### For Android
1. **Google Service Account Key** (`google-service-account.json`)
   - Create in Google Cloud Console
   - Grant "Play Android Developer API" permission
   - Place in project root

2. **Keystore** (auto-generated by EAS or provide custom)

### For iOS
1. **Apple ID** in `eas.json`
2. **App Store Connect App ID**
3. **Distribution Certificate** (auto-managed by EAS)
4. **Provisioning Profile** (auto-managed by EAS)

## 🆘 Troubleshooting

### Common Issues

**Build Fails:**
- Check all required files exist
- Verify package names/bundle IDs
- Ensure all dependencies are installed
- Check EAS Build logs

**App Crashes on Launch:**
- Verify Firebase configuration
- Check native module linking
- Test on real devices
- Review crash logs

**Rejection from Stores:**
- Review store-specific guidelines
- Ensure privacy policy is accessible
- Verify all permissions are justified
- Test all features thoroughly
