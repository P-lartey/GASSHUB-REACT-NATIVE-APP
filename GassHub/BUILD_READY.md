## 🚀 Building the App

### Step 1: Install EAS CLI (if not already installed)
```bash
npm install -g eas-cli
```

### Step 2: Login to Expo/EAS
```bash
eas login
```

### Step 3: Configure EAS Build
```bash
eas build:configure
```

### Step 4: Build for Production

#### Option A: Build Both Platforms
```bash
# Android (Google Play Store)
eas build --platform android --profile production

# iOS (App Store) - Run separately
eas build --platform ios --profile production
```

#### Option B: Build Android Only (Recommended First)
```bash
eas build --platform android --profile production
```

### Step 5: Submit to Stores

After build completes:

#### Android Submission:
```bash
eas submit --platform android --latest
```
OR manually:
1. Download the .aab file from EAS dashboard
2. Go to [Google Play Console](https://play.google.com/console)
3. Create new app → Upload to Production track
4. Fill store listing → Submit for review

#### iOS Submission:
```bash
eas submit --platform ios --latest
```
OR manually:
1. Build will be uploaded to App Store Connect automatically
2. Go to [App Store Connect](https://appstoreconnect.apple.com)
3. Select your app → Select the build
4. Fill app information → Submit for review
